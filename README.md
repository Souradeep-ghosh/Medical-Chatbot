# 🏥 Medical Chatbot

A RAG-based Medical Chatbot built with LangChain, Pinecone, and OpenAI GPT-4o.

## 🚀 Live Demo
👉 **[Click here to try the chatbot](https://huggingface.co/spaces/Sg31Ghosh/medical-chatbot)**

## 🛠️ Tech Stack
- **LLM:** OpenAI GPT-4o
- **Embeddings:** HuggingFace sentence-transformers/all-MiniLM-L6-v2
- **Vector Store:** Pinecone
- **Framework:** LangChain
- **Backend:** Flask
- **Deployment:** Hugging Face Spaces (Docker)

## 📌 Features
- RAG (Retrieval Augmented Generation) pipeline
- Medical knowledge base from PDF documents
- Conversational memory
- Clean chat UI
